# 📊 Power BI Project 2

## 📌 Overview
This repository contains a Power BI dashboard project designed to analyze and visualize data effectively. The report provides meaningful insights through interactive visuals, helping users make data-driven decisions.

## 🧾 Project File
- `Power BI-Project 2.pbix` – Main Power BI report file

## 🎯 Objectives
- Transform raw data into meaningful insights  
- Create interactive dashboards for better understanding  
- Enable easy filtering and drill-down analysis  

## 📊 Features
- Interactive visualizations  
- Dynamic filtering and slicers  
- Clean and user-friendly dashboard layout  
- Data-driven insights for decision-making  

## 🛠️ Tools & Technologies
- Microsoft Power BI  
- Data Modeling  
- Data Visualization Techniques  

## 🚀 How to Use
1. Download the `.pbix` file from this repository  
2. Open it using **Power BI Desktop**  
3. Refresh data if needed  
4. Explore the dashboard using filters and visuals
